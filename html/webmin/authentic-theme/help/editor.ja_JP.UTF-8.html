エディタのキーボードショートカット
検索/置換
キープレス コマンド
Ctrl + F 検索
Ctrl + H 置換
Ctrl + ⇧ + R すべてを置換
Ctrl + G Find next
Ctrl + ⇧ + G 前の検索
Alt + G 行にジャンプ
テキスト操作
キープレス コマンド
Meta + ⇧ + W Toggle soft wrap
Ctrl + KU 大文字に変換
Ctrl + KL 小文字に変換
編集
キープレス コマンド
Ctrl + Space オートコンプリートを有効にする
Ctrl + X 切り取り
Ctrl + ⇧ + ↑ 行/選択範囲を上に移動
Ctrl + ⇧ + ↓ 行/選択を下に移動
Ctrl + L 行を選択 - 次の行を選択するために繰り返します。
Ctrl + D 単語を選択 - 他のものを繰り返し選択
Ctrl + M 閉じ括弧にジャンプ繰り返して括弧を開きます
Ctrl + ⇧ + M 現在のかっこのすべての内容を選択する
Ctrl + ⇧ + K 行の削除
Ctrl + KK カーソルから行末までの削除
Ctrl + K + Backspace 行の先頭からカーソルから削除
Ctrl + ⇧ + D 重複する行
Ctrl + J 下の行を現在の行の末尾に結合する
Ctrl + / 現在の行のコメント/コメント解除
Ctrl + ⇧ + / コメントの現在の選択をブロックする
Ctrl + Y やり直し、または最後のキーボードショートカットコマンドの繰り返し
Ctrl + ⇧ + V 正しくペーストしてインデントする
Ctrl + Space 次のオートコンプリート提案を選択
Ctrl + U ソフト元に戻す。繰り返したときに元に戻す前に最後の変更にジャンプします
